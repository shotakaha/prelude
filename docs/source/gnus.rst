Gnusの使い方
==================================================


Summary バッファの表示
==================================================

* ``gnus-summary-line-format`` で設定できる
* デフォルトは ``"%U%R%z |%d| %I%(%[%4L: %-23,23f%]%) %s\n"`` らしい
* これだと **日付がない** ので不便
* なので ``%d`` を挿入することにする

  * ``%U``, ``%R``, ``%z`` はいじらないほうがいいらしい
  * ``%I`` はリプライ数によって深さが変わるインデント
  * 日付は場所固定がいいので ``%I`` の前にした
  * ついでに ``|%d|`` として区切りを入れた

.. code-block:: emacs

   (setq gnus-summary-line-format  "%U%R%z |%d| %I%(%[%4L: %-23,23f%]%) %s\n")
